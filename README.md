# Movie-and-Product-Review-Sentiment-Analysis
A machine learning project that classifies movie and product reviews as positive or negative using Natural Language Processing (NLP) and Scikit-lea
## 📌 Project Overview
This project uses Machine Learning and Natural Language Processing (NLP) to classify movie and product reviews as either **Positive** or **Negative**.

## 🚀 Features
- Sentiment classification
- Text preprocessing
- TF-IDF feature extraction
- Multiple machine learning models
- Model evaluation using accuracy, precision, recall, and F1-score

## 🛠️ Technologies Used
- Python
- Jupyter Notebook / Google Colab
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib

## 📂 Project Files
- `Sentiment_Analyzer_project.ipynb` – Main notebook
- `README.md` – Project documentation
- `requirements.txt` – Required Python libraries

## ▶️ How to Run
1. Clone the repository.
2. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
3. Open `Sentiment_Analyzer_project.ipynb` in Jupyter Notebook or Google Colab.
4. Run all cells.

## 📊 Results
The model predicts whether a movie or product review has a positive or negative sentiment using machine learning techniques.
